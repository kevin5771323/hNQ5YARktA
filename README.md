# 项目简介

## 核心功能
- 基于YOLOv8的纽扣电池缺陷检测系统，可识别['dirty', 'depression', 'scratch']三类缺陷
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供UI界面，实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和IOU阈值
- 可选择显示标签和原图
- 提供批量检测功能，可直接查看所有检测结果
- 支持检测结果保存
- 内含1000张标注数据集和训练好的模型

## 适用场景
- 纽扣电池生产线质量检测
- 电池制造企业自动化质检流程
- 电池缺陷研究与分类
- 电池质量控制系统

## 技术特点
- 采用YOLOv8目标检测算法
- 使用PyQt5构建用户界面
- 环境配置：python3.9, opencv-python, PyQt5, ultralytics, torch1.9等
- 提供完整源码、数据集、训练结果和运行教程
- 支持图片、视频及摄像头多种输入方式
